## Program 17: Write a program using double.
```
#include<stdio.h>
int main()
{	
float age1;
double age2;
printf("Please enter your age:");
scanf("%f",&age1);
printf("Enter your child age:");
scanf("%lf",&age2);
printf("age1= %f",age1);
printf("\nage2= %lf",age2);
return 0;
}
```
**Output**
```
Please enter your age:40
Enter your child age:20
age1= 40.000000
age2= 20.000000
```
